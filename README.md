これは、WordPressとNextJSで構築された指紋ブラウザソフトウェアの公式サイトです。バックエンドはWordPress REST APIを搭載し、twentytwentytwoテーマをベースにしています。フロントエンドはNextJS、TailwindCSS、そしていくつかのNode.jsライブラリで構築されています。
詳しくは lalicat.com をご覧ください。
